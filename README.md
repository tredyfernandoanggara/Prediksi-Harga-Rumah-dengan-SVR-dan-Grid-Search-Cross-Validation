# Prediksi-Harga-Rumah-dengan-SVR-dan-Grid-Search-Cross-Validation

Repositori ini berisi proyek Machine Learning untuk memprediksi harga rumah berdasarkan berbagai fitur yang relevan, menggunakan Support Vector Regression (SVR). Untuk mendapatkan hyperparameter terbaik, dilakukan tuning menggunakan Grid Search Cross Validation.
